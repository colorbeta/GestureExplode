# 关于f和t识别错误的问题
Google 的 ML Kit 引擎是基于欧美用户的海量书写习惯训练的。由于底层 AI 模型不可修改，目前最有效的物理解决办法是：在写 f 的时候，刻意夸张一下它顶部那个向右的“大弯钩”，让 AI 从几何形状上无法把它和直直的 t 混淆；写 t 的时候尽量保持竖线笔直，或者底部带点向右的小尾巴。

# 关于参数设置的搜索
谷歌从未提供过类似 getAllSettingsMenus() 这样的接口。安卓的“设置”本质上是一个包名为 com.android.settings 的普通 App，目前仅支持字典搜索如下参数：蓝牙设置,移动网络,数据使用情况,NFC 设置,飞行模式,投屏设置,VPN设置,显示设置,声音与振动,壁纸设置,日期和时间,语言和输入法,位置信息,安全设置,隐私设置,生物识别/指纹,锁屏设置,应用管理,默认应用,存储设置,账号与同步,关于手机,系统更新,开发者选项,电池信息,省电模式,无障碍,搜索设置,勿扰模式
